# C# Stopwatch Project

This is a basic C# console application implementing a Stopwatch.

## Features
- Start, Stop, and Reset the timer
- Display elapsed time in hh:mm:ss format
- Simple and clean console interface

## Purpose
This project was created as the **first exercise** in a C# course by Hamid Reza Garanmaye.

## Usage
1. Open the project in Visual Studio.
2. Run the application.
3. Use the menu to Start, Stop, or Reset the stopwatch.
